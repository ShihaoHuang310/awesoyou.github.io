Java环境配置

Author: huang shh Date: 九月 29, 2024  20:45:18 Category: 开发配置

鼠标右键打开—>电脑属性—>选择 高级系统设置

1、新建变量名：JAVA_HOME，变量值

C:\Program Files\Java\jdk1.8.0_311

2、新建变量名：CLASSPATH，变量值

.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar

3、配置好前面两项后在Path添加环境变量

%JAVA_HOME%\bin

最后所有环境变量都配置好后，键盘输入win + r键打开 cmd 命令窗口：分别输入java|java --version|javac

java

java –version

javac

如果以上都有显示则环境安装成功！😁
